PTViewerNG - Plugin-Free Open-GL Panorama Viewer

PTViewerNG is based on WebGL. To view the demo download and install one of the compatible browsers. I have tested the viewer with Firefox (the nightly built pre-alpha version is required). Make sure to enable WebGL as described here.

High resolution example courtesy Erik Leeman.
Caution: 6 MB, no download feedback.

Accelerate left-right-up-down using arrow keys, or Numpad 4,6,8,2.
Zoom in and out using +/- keys.
Stop using 'c' or space or Numpad 5.
